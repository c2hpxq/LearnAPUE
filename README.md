# 习题源代码使用方法

## 编译代码

在主目录下运行

```bash
make
```

如果只想编译某一章的代码，例如Chapter10那么

```bash
cd Chapter10; make
```

## 运行代码

以Chapter10 exercise5为例

```bash
cd Chapter10
make run-ex10.5
```
